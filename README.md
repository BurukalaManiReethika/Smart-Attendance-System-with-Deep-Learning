# Smart Attendance System using Deep Learning

## 📌 Project Overview
This project is a Smart Attendance System that uses Deep Learning and Face Recognition to automatically mark attendance in real time.

## 🧠 Technologies Used
- Python
- OpenCV
- CNN (Deep Learning)
- NumPy
- Pandas

## ⚙️ How It Works
1. Captures live video using webcam
2. Detects faces using OpenCV
3. Recognizes faces using trained deep learning model
4. Matches with stored dataset
5. Records attendance with date and time in CSV file

## ✨ Features
- Real-time face detection
- High accuracy recognition
- Automated attendance marking
- Saves attendance data securely

## 📂 Project Structure
```
smart-attendance-system/
│
├── dataset/
├── model/
├── attendance.csv
├── main.py
├── train_model.py
└── requirements.txt
```

## 🚀 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run training:
   python train_model.py

3. Run system:
   python main.py
